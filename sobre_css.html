Sobre CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.

Como funciona CSS?

O CSS é uma ferramenta muito potente que possibilita criar diversas funcionalidades ao invés de usar JavaScript ou outra linguagem mais pesada. Se usado com moderação, CSS pode viabilizar uma ótima experiência ao desenvolvedor e usuários das páginas web.

o Cascading Style Sheets é possível criar animações complexas, criar efeitos com uso de parallax, que faz parecer que a imagem de fundo tem uma profundidade diferente um dos outros, criar sites interativos e também jogos com HTML5 e CSS3.

Propiedades do CSS

  1. background: Cor de fundo, imagem, transparência.
  2. background-attachment: Rolagem do fundo / marca d’água.
  3. background-image: Imagem de fundo.
  4. background-color: Cor de fundo ou transparência.
  5. background-position: Posicionamento da imagem de fundo.
  6. background-repeat: Configuração lado-a-lado da imagem de fundo.
  7. border-bottom: Largura, estilo e cor da borda inferior.
  8. border-bottom-color: Cor da citada borda.
  9. border-bottom-style: estilo da citada borda.
  10. border-bottom-width: largura da citada borda
